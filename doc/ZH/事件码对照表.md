## 数据与计算节点共用

| 事件码  |                             描述                             |
| :-----: | :----------------------------------------------------------: |
| 0008000 | 成功终止符，成功终止符，数据节点或计算节点只有最终运行成功，才会生成该事件 |
| 0008001 |  失败终止符，数据节点或计算节点任何运行失败，都会生成该事件  |
